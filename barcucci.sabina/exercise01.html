Videoconference

Fab Academy 2015 global meeting number one has took place January the 28th, at 3 pm for me in Italy. I used XMeeting as client software to connect with the MCU since Jabber Video doesn't work on my operating system release (which is OS X Snow Leopard).
Apparently everything worked quite well here, except that the Neil's screen sharing wasn't really visible for me using XMeeting. I attach a screenshot of the Xmeeting video interface to show how small was the section of Neil's screen visualization. I am trying to solve the issue testing other client softwares for MCU and I am also still evaluating pros and cons to eventually upgrade my OS X to Yosemite in order to easily use Jabber Video. Let's see.

Web development

To start the website development, I first watched this thorough lecture of Massimo Menichinelli, my local tutor, on HTML and CSS



Afterward, in our lab session at Wemake MakerSpace in Milan, we got warmed with the text editor Bracket to edit a pre-existing css + html template based on Bootstrap and we started filling the template with contents previously produced as the personal resume and the conceptual ideas on a potential final project.

Version control

The second part of the lab session at Wemake focused on version control. We installed Git on our computer in order to use GitHub and also the mobile version for Android to check updates and pull requests everywhere. We work on our local organization repository at this link https://github.com/WeMakecc/FabAcademy2015-WeMake



I attach some precious resources about github procedure to add, commit and push files on the online repository

http://supsi.openp2pdesign.org/2014-2015/Design-with-the-user_Slides/03_27Nov14.html#/
