Exercise 14 - Interface and Application Programming

Assignment

Write an application that interfaces with an input and/or output device


Making a DIY Bluetooth module

I wanto to create a bluetooth module for my final project and test how it works detecting data from a sonar. In parallel I will try to create a basic GUI. Coding and programming is my real “Achilles heel”. I am totally a beginners. Python and processing seems to me almost black box. The final scope of this exercize is to connect bluetooth module and a Max Sonar sensor with a basic GUI. Basic ideas is to create a bluetooth board using a MH-10 Bluetooth modules and soldering it on a DIY board specifically designed to be connected to a system. Before to make it, I studied some datasheets and search on the Web some tutorials. I found the following information and projects: Bluetooth BTH 05
Bluetooth with freedom board
Bluetooth to serial adapter


It has been very complex to design this kind of board using Eagle and/or KiCad. In particular it is very difficult to find schematics on board about the MH-10 Bluetooth Module. So I conceived an unconventional activity: I re-designed the Bluetooth board using Illustrator. Looking datasheets and using the graphic elements of the Hello Echo board (pin heard, resistors, led, and button) I designed a board that can be host a bluetooth module in order to use it. Finally, I recreated the same board from zero because the libraries for Bluetooth are not so developed.
