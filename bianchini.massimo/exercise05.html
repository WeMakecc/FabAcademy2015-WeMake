Exercise 05 - 3D scanning and printing

Assignment

Design and 3D print an object (small, few cm) that could not be made subtractively. 3D scan an object (and optionally print it)


3D printing

For this exercise I will try to 3D print the case of my object/project. I already prepared a first version of a 3D model in Rhino for Exercise 01. In these days I am working to launch Polifactory, the new makerspace of Politecnico di Milano (it will open 19th march). In these days I am setting up a new 3D printer, a WASP PowerEvo. For this reason, I need one/two days more to complete this task and 3D printing these two objects. Moreover, for Polifactory Open day we are started to prepare a small exhibition combining arts and 3D printing. We are choosing/selecting bottles from famous painting (e.g. Morandi, Cezanne, Picasso,) and we want to create 3D models to be 3D printed.

3D scanning and printing

For this exercise, I tried to 3D scan three different objects using 123Dcatch: A green chair, an orange electrotool (black&decker) and a common plug.

  • The green Chair: a complete failure. Maybe this object is to big to 3D scan and it can be confused with the environment (green and beige). The final 3D scan results partial and fragmented. It is not possible complete the object and then 3D print it creating mesh.
  • The electrotool: another failure. The final result of this process is a partial 3D model. Some parts are well detailed, other completely irregular. I am not sure but I think this object is to complex to be 3D scanned using 123DCatch.
  • The plug: a partial success. This is a small object, very easy capture images of it. For this reason the process generates a primitive, irregular but almost complete 3D model that can be opened in 3D software to be modified and then prepared for 3D printing in the next days
